# Changelog

All notable changes to `filament-theme-switcher` will be documented in this file.

## v1.0.0 - 2024-01-01

### Added
- Initial release
- 7 pre-built themes: Default, Sunset, Ocean, Forest, Midnight, Rose, Amber
- Theme settings page with visual theme selector
- Custom color picker for theme customization
- Global and per-user theme modes
- Middleware for applying themes
- Livewire theme switcher dropdown component
- Full translation support
- Authorization support via `canViewThemesPage()`
- Filament v3.x and v4.x compatibility
